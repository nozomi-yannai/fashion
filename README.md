# README

## 開発言語

- Ruby 3.3.0
- Rails rails 6.1.7.6

## 実行手順

$ git clone git@github.com:nozomi-yannai/fashion.git
$ cd fashon
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
